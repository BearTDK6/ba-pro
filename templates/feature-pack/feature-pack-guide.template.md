# Feature Pack Guide

Use this as the standard internal shape for a new feature folder:

```text
features/<domain>/<feature>/
├── 01_discovery/
├── 02_requirements/
├── 03_analysis/
├── 04_solution/
├── 05_uat/
├── 06_delivery/
├── 07_decisions/
└── assets/
```

Recommended minimum files:

- `01_discovery/<date>-project-brief.md`
- `02_requirements/<date>-requirements.md`
- `03_analysis/<date>-analysis.md`
- `03_analysis/<date>-rtm.csv`
- `04_solution/<date>-solution-outline.md`
- `05_uat/<date>-uat-plan.md`
- `06_delivery/<date>-release-readiness.md`
- `07_decisions/<date>-decision-log.md`
