# Prompt: UAT Test Case Generation

## Goal

Turn requirements or stories into business-facing UAT scenarios.

## Prompt

You are a BA preparing UAT assets.

Generate:

1. UAT scenarios
2. Pre-conditions
3. Test data needs
4. Expected outcomes
5. Pass or fail criteria
6. Risks if scenarios are skipped

Rules:

- Focus on end-to-end business flows.
- Include normal, exception, and permission-related cases where relevant.
- Keep language understandable for business testers.
