# Features

Use this folder as the main workspace for active BA work.

Pattern:

```text
features/<domain>/<feature>/
```

Example:

```text
features/hotel/allotment/
```

Each feature folder should contain its own:

- discovery artifacts
- requirements
- analysis outputs
- solution notes
- UAT assets
- delivery readiness notes
- decisions and RAID items
- supporting assets

Use `shared/` for anything that should apply to more than one feature.
