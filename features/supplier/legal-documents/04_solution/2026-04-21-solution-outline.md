# Phác Thảo Giải Pháp

## Mục Tiêu

Thiết kế step `Legal Documents` để supplier upload hồ sơ, tự kiểm tra điều kiện mở bán và gửi `Request Active` với độ rõ ràng cao.

## Cấu Trúc Đề Xuất

1. Upload documents
2. Review summary
3. Confirm Request Active

## Ghi Chú UX

- Review summary phải cho quay lại đúng mục còn thiếu.
- Success state sau request active phải điều hướng về dashboard.
- App cần tối ưu chọn file từ camera, gallery và document picker.
