# Phác Thảo Giải Pháp

## Mục Tiêu

Cho phép supplier tạo room type nhanh, đúng dữ liệu và đủ điều kiện để cấu hình rate.

## Cấu Trúc Đề Xuất

1. Room list
2. Add / Edit Room form
3. Clone room action

## Ghi Chú UX

- Room list nên hiển thị trạng thái hoàn tất của từng room.
- Clone room nên là action trực tiếp từ danh sách room.
- Validation capacity nên hiển thị ngay tại field.
